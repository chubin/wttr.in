Использование:

    $ curl wttr.in          # погода в текущем местоположении на основе вашего IP
    $ curl wttr.in/svo      # погода в аэропорту Шереметьево (код ICAO: SVO)
    $ curl wttr.is          # эквивалентно "curl wttr.in"

Поддерживаемые типы местоположений:

    /paris                  # название города
    /Старый+Оскол           # название города из 2-х слов (+ для пробела)
    /Eiffel+tower           # любое местоположение, например "Эйфелева башня" (+ для пробела)
    /Москва                 # название любого местоположения в Юникоде на любом языке
    /moon+US                # +US для города Moon в США
    /moon+France            # +France для города Moon во Франции
    /muc                    # код аэропорта ICAO (3 буквы)
    /@stackoverflow.com     # доменное имя
    /94107                  # почтовый индекс (только для США)
    /-78.46,106.79          # GPS-координаты

Специальные условные местоположения:

    /moon                   # Фаза Луны
    /moon@2016-10-25        # Фаза Луны для указанной даты (@2026-12-25) (@ГГГГ-ДД-ММ)

Единицы измерений:

    ?m                      # международная метрическая система (СИ) (используют везде, кроме США)
    ?u                      # USCS (используют в США)
    ?M                      # скорость ветра в м/с

Опции отображения:

    ?0                      # только текущая погода
    ?1                      # погода сегодня + 1 день
    ?2                      # погода сегодня + 2 дня
    ?d                      # ограничить вывод стандартными символами консольного шрифта
    ?n                      # узкая версия (только день и ночь)
    ?q                      # тихая версия (без текста "Прогноз погоды")
    ?A                      # игнорировать User-Agent и принудительно использовать формат вывода ANSI (терминал)
    ?F                      # не отображать строку "Следуйте"
    ?Q                      # сверхтихая версия (без "Прогноз погоды", нет названия города)
    ?T                      # отключить терминальные последовательности (без эффектов и цвета)

PNG-опции:

    /paris.png              # сгенерировать PNG-файл
    ?p                      # добавить рамочку вокруг
    ?t                      # прозрачность = 150
    transparency=...        # прозрачность от 0 до 255 (255 = не прозрачно)

Опции можно комбинировать:

    /Paris?0pq              # символ " ? " является разделителем
    /Paris?0pq&lang=fr      # символы " ? " и " & " являются разделителями
    /Paris_0pq.png          # в PNG-запросах опции разделяются знаком подчёркивания "_"
    /Rome_0pq_lang=it.png   # длинные опции PNG-запросов разделяются знаком подчёркивания "_"

Альтернативный домен:

    $ curl wttr.is          # wttr.is является полностью эквивалентным резервным доменом для wttr.in
    $ curl wttr.is/London   # Вы можете использовать его в качестве полноценной замены

Локализация:

    $ curl ru.wttr.in/Москва
    $ curl wttr.in/москва?lang=ru
    $ curl -H "Accept-Language: ru" wttr.in/москва
    $ curl fr.wttr.in/Paris
    $ curl wttr.in/paris?lang=fr
    $ curl -H "Accept-Language: fr" wttr.in/paris

Поддерживаемые языки:

    FULL_TRANSLATION (поддерживаются)
    PARTIAL_TRANSLATION (в процессе)

Специальные страницы:

    $ curl wttr.in/:help    # показать справку на английском языке
    $ curl ru.wttr.in/:help # показать справку на русском языке
    или
    $ curl wttr.in/:help?lang=ru
    /:bash.function         # показать рекомендованную функцию wttr()
    /:translation           # показать список переводчиков wttr.in

Примеры:

    $ curl wttr.in/Москва?M\&lang=ru    # отобразить погоду в Москве на ближайшие 3 дня,
                                        # ветер в м/с (?M), на русском языке (&lang=ru)

Для упрощения и быстрого набора команды в терминаде, в "Домашнем каталоге"
пользователя создаётся файл ".bash_aliases " для команд-псевдонимов
/home/имя_юзера/.bash_aliases
со следующим содержанием:
============================================
# alias.sh

alias wttrm='curl wttr.in/Москва?M\&lang=ru'
# ветер в м/с (?M)
# или
alias wttrk='curl wttr.in/Москва?lang=ru'
# ветер в км/ч
# замените Москва на свое местоположение
============================================

После этого для получения погоды в терминале будет достаточно набрать команду:

wttrm или wttrk
